# Senior Project: Phishing Detector

This project is a phishing detection tool designed to analyze URLs and identify potential phishing websites.

The system will allow users to enter a URL and determine if the website may be malicious based on different security indicators.

Features (Planned):
- URL analysis
- Suspicious keyword detection
- HTTPS security check
- Risk scoring system
- User-friendly web interface

Technologies:
- HTML
- CSS
- JavaScript
- Node.js (backend)
- GitHub for version control
